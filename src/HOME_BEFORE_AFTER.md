**AFTER:**

### **Tab Switching:**

**BEFORE:**
```
User clicks "View All Wishes"
  ↓
Navigate to new screen
  ↓
Fetch all wishes
  ↓
Render new page
```
**Time:** ~2 seconds

**AFTER:**
```
User clicks "Wishes" tab
  ↓

Render in same page
```
