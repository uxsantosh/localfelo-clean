## 🚀 Quick Create

### 1. favicon.ico
- Export `/assets/logo.svg` as 32×32 PNG
- Convert to ICO: https://favicon.io/
- Save to `/public/favicon.ico`

### 2. og-image.png
- Use Canva: https://www.canva.com/
- Size: 1200×630
- Green background + logo + text
- Save to `/public/og-image.png`

### 3. twitter-image.png
- Same as og-image but 1200×600
- Save to `/public/twitter-image.png`

### 4. logo.png
- Export `/assets/logo.svg` as 512×512 PNG
- Save to `/public/logo.png`
