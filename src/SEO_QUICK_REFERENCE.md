## 🆘 Troubleshooting

### Issue: Site not indexed after 1 week
**Solution**: 
1. Check robots.txt isn't blocking
2. Submit URL in Search Console
3. Check for crawl errors
4. Verify sitemap submitted

### Issue: Poor page speed score
**Solution**:
1. Optimize images (WebP format)
2. Enable compression
3. Minimize JavaScript
4. Use CDN for assets

### Issue: Structured data errors
**Solution**:
1. Validate with Schema validator
2. Check JSON syntax
3. Ensure all URLs are absolute
4. Update image references
