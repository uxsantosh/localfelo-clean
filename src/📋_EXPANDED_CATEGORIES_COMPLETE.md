## 🎯 CATEGORY FEATURES

### Each Category Has:
- **Emoji** - Visual identification
- **Name** - Clear, simple title
- **Description** - What it covers
- **Keywords** - For AI auto-detection (5-15 keywords each)
- **Sub-skills** - Detailed service options (for helpers)
- **Popular flag** - Mark high-demand categories

### Example - Cooking Category:
```javascript
{
  name: 'Cooking',
  emoji: '🍳',
  description: 'Home cooking, tiffin, meal prep, catering',
  keywords: ['cook', 'cooking', 'food', 'meal', 'tiffin', 'lunch', 
             'dinner', 'breakfast', 'chef', 'recipe', 'cuisine'],
  subSkills: [
    'Home cooking',
    'Tiffin service',
    'Meal preparation',
    'North Indian cuisine',
    'South Indian cuisine',
    'Chinese food',
    'Breakfast/lunch/dinner',
    'Special occasion cooking',
  ],
  popular: true
}
```

## 🚀 HOW AUTO-DETECTION WORKS

### Smart Keyword Matching:
```
User types: "Need tiffin service for lunch daily"
           ↓
Keywords detected: "tiffin", "lunch", "daily"
           ↓
Best match: 🍳 Cooking (score: 15 points)
           ↓
Auto-selected: "Cooking"
```

### Scoring System:
- **Exact word match**: +5 points
- **Partial match**: +2 points
- **Best score wins**: Category with highest score selected
