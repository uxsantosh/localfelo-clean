========================================
IMPORTANT NOTES
========================================

1. NO emojis anywhere
2. Use "Rs" not "₹" symbol
3. Category AUTHENTICATION = free, MARKETING = requires user opt-in
4. Replace localfelo.com with your actual domain if different
5. All variables are type: TEXT
6. Variable names in {{}} must match exactly when sending
7. Approval takes 1-24 hours per template
8. Start with Template 1 (OTP) first - it's simplest
