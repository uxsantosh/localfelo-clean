# Security Policy

This is the security.txt file for LocalFelo.
We welcome security researchers to report vulnerabilities responsibly.

# Scope

In scope:
- Web application vulnerabilities (XSS, CSRF, SQLi, etc.)
- Authentication and authorization issues
- Data exposure or privacy concerns
- API security issues

Out of scope:
- Social engineering attacks
- Physical security issues
- Denial of Service (DoS) attacks
- Issues in third-party services

# Disclosure Policy

1. Email security@localfelo.com with details
2. Allow us 90 days to address the issue
3. Do not publicly disclose until we confirm the fix
4. We will credit you (if desired) after resolution

# Safe Harbor

LocalFelo commits to:
- Not pursue legal action for good faith security research
- Work with researchers to understand and resolve issues
- Recognize and thank researchers publicly (with permission)

Thank you for helping keep LocalFelo safe!
